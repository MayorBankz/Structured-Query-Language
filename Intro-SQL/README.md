# TryHackMe - Introduction to Structured Query Language (SQL)
* Date: 24-11-2025

## What is SQL?
SQL (Structured Query Language) is a computer language used to talk to databases.

A <em>database</em> is a place where information is stored, organized, and managed.

SQL helps you:
* Store data
* Find data
* Update data
* Delete data
* Organize data

## What SQL is used for?
SQL is used to:
1. Get information from a database
Example:
* "Show all customers"
* "Show sales for last week"

2. Add new information
Example
* Add a new user or product

3. Update Information
Example: Change someone's phone number

4. Delete information
Example: Remove an old record

5. Create and manage tables
Example: Tables are like spreadsheets inside a database


## How SQL Works 
A database has tables - rows and columns, just like Excel
